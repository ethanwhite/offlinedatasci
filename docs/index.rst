Welcome to offlinedatasci's docs!
=================================

Data science tools and educational materials are predominantly distributed online,
making them difficult to access in locations or communities with limited internet connectivity.
This internet dependency can be mitigated by downloading materials when/where internet is available,
but this can be difficult to manage for training opportunities like workshops and for people new to data science.
*offlinedatasci* supports teaching and doing data science in internet-limited environments
by downloading and configuring common tools and data science lessons including:

* Installers for data science languages: Currently R and Python
* Installers for common data science IDEs: Currently RStudio
* Data science focused local mirrors of package repositories: Currently CRAN and PyPI
* Locally browseable clones of data science teaching websites: Currently The Carpentries

offlinedatasci is part of `CarpentriesOffline <https://carpentriesoffline.org>`_, a broader set of
tools and instructional materials for teaching and practicing data science in internet-limited environments.
